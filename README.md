# 📖 About Project

**TwmApi26** is a RESTful Web API developed using **ASP.NET Core**, **ADO.NET**, and **SQL Server**.

The project demonstrates complete CRUD (Create, Read, Update, Delete) operations using direct SQL queries without Entity Framework.

---

# ✨ Features

- ✅ Create Student
- ✅ Get All Students
- ✅ Get Student By Id
- ✅ Update Student
- ✅ Delete Student
- ✅ SQL Server Integration
- ✅ ADO.NET Connectivity
- ✅ Swagger Documentation
- ✅ RESTful API
- ✅ JSON Response

# 📡 API Endpoints

| Method | Endpoint | Description |
|----------|----------------|----------------|
| GET | /api/student | Get All Students |
| GET | /api/student/{id} | Get Student By Id |
| POST | /api/student | Add Student |
| PUT | /api/student/{id} | Update Student |
| DELETE | /api/student/{id} | Delete Student |
